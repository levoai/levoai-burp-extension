This extension helps you to create OpenApi specifications for your APIs based on the traffic captured by Burp.

This extension needs Levo's Satellite service to running at localhost:9999. See Levo docs for instructions on how to run Satellite with docker compose.

Once you have the Satellite service, this extension sends all the JSON based HTTP request and responses to Levo's Satellite, which builds specs and uploads them to Levo SaaS. All your data is dropped on the Satellite host itself and not sent to Levo's SaaS.

There are the following configuration options in the menu bar:

  • Send only requests from defined target scope.
  • Pause sending the traffic to Levo's Satellite.
